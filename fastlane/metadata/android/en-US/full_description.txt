LastQuakes is a comprehensive application providing real-time global earthquake monitoring with multi-source data integration, customizable push notifications, interactive map visualization, and advanced filtering capabilities.

<b>Core Features:</b>

<ul>
<li><b>Multi-Source Data Integration</b>:
<ul>
<li><b>USGS (U.S. Geological Survey)</b>: Comprehensive global earthquake data.</li>
<li><b>EMSC (European-Mediterranean Seismological Centre)</b>: Enhanced coverage for European and Mediterranean regions.</li>
<li><b>Intelligent Deduplication</b>: Automatic removal of duplicate events from multiple sources.</li>
</ul>
</li>

<li><b>Interactive Map Visualization</b>:
<ul>
<li>High-performance map rendering with smooth panning and zooming.</li>
<li>Multiple Base Layers: Street Map, Satellite Imagery, Terrain View, Dark Theme.</li>
<li>Magnitude-Based Markers: Color-coded markers with size scaling.</li>
<li>Fault Line Overlays: Optional tectonic plate boundary visualization.</li>
</ul>
</li>

<li><b>Dynamic List View</b>:
<ul>
<li>Filter by magnitude, country/region, and distance.</li>
<li>Sort by time, magnitude, or distance.</li>
<li>Modern card-based design with gradient headers.</li>
</ul>
</li>

<li><b>Detailed Event Analysis</b>:
<ul>
<li>Magnitude, depth, location coordinates, and tsunami warning status.</li>
<li>Distance from user's location.</li>
<li>Interactive map pinpointing event location.</li>
</ul>
</li>

<li><b>Statistics &amp; Analytics</b>:
<ul>
<li>Visual representation of seismic activity trends.</li>
<li>Magnitude and geographic distribution charts.</li>
</ul>
</li>

<li><b>Widget</b>:
<ul>
<li>Add widget to home screen to display latest earthquakes.</li>
</ul>
</li>

<li><b>Privacy &amp; Security</b>:
<ul>
<li><b>FOSS Version</b>: Completely open-source with NO proprietary dependencies.</li>
<li><b>Secure Storage</b>: Encrypted storage for sensitive data.</li>
</ul>
</li>
</ul>

<b>Note</b>: This is an independent project and is not officially affiliated with USGS, EMSC, or any government seismological organization.
